Embed Instructions

Paste the snippet below into any HTML page to embed the FRaaS kiosk directly. The iframe auto-sizes and requires camera permission from the user's browser.

Step 1 — Embed tag
HTML
<iframe
  src="https://app.facesync.io/kiosk/org_527df553"
  width="100%"
  height="600"
  frameborder="0"
  allow="camera"
  title="FRaaS Attendance Kiosk"
></iframe>
Step 2 — Add camera permission header
Permissions-Policy: camera=(self "https://app.facesync.io")
💡 Notes:
• The iframe requests camera access from the user automatically.
• Works on Chrome, Edge, and Firefox. Safari requires user gesture.
• All recognition data is sent to your FRaaS account — no data is stored in the iframe host page.
